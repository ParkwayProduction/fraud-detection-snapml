# [ MODULE : FRAUD_DETECTION_SNAPML ]

### [ MISSION_OVERVIEW ]
High-speed binary classification pipeline utilizing the European cardholder dataset to identify fraudulent transactions. This module focuses on the tactical application of accelerated machine learning in high-stakes financial environments.

### [ ANALYTICAL_STRIKE_FORCE ]
* **Snap ML Acceleration:** Implementation of IBM Snap ML to achieve up to 4.7x speedup in Decision Tree and SVM training.
* **Imbalance Management:** Strategic application of `compute_sample_weight` to ensure accurate detection in extremely rare fraudulent events (0.172% rate).
* **Metric Validation:** Rigorous performance assessment using ROC-AUC and Hinge Loss metrics.
