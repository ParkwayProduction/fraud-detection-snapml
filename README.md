# fraud-detection-snapml
Advanced classification system for credit card fraud detection utilizing Scikit-Learn and IBM Snap ML for high-performance CPU/GPU acceleration

[ MODULE : FRAUD_DETECTION_PIPELINE ]
[ MISSION_OVERVIEW ]
Deployment of a high-speed binary classification pipeline to detect fraudulent financial transactions. This module utilizes the European cardholder dataset and implements high-performance acceleration via the IBM Snap ML library.

[ ANALYTICAL_STRIKE_FORCE ]
Decision Tree Classifier: Optimized for real-time inference with 4.7x acceleration.

Support Vector Machine (SVM): Fine-tuned with hinge loss for robust class separation.

Data Normalization: Pre-processing via L1-norm and Standard Scaling protocols.
